## 1. 개요

이 노트북의 핵심 목적은 다음 두 ETF에 **동일한 조건**으로 투자했을 때 어느 쪽이 더 유리한지를 과거 데이터 기반으로 비교하는 것입니다.

|-----|------|------|

## 5. 투자 시뮬레이션

### 시뮬레이션 파라미터

| 항목 | 값 |
|------|----|
| 초기 투자금 | 10,000,000 KRW (1,000만 원) |
| 월 인출금 | 120,000 KRW (12만 원) |
| 배당 처리 | 재투자 |
| 인출 시점 | 매월 마지막 거래일 |

### `simulate_single_etf_investment()` 함수 로직

```
[시작] 첫날 종가로 초기 주식 수 계산
  etf_shares = initial_investment / 첫날_종가

[매일 반복]
  ① 배당 처리 (재투자)
       배당금 발생 시 → 해당 금액으로 주식 추가 매수

  ② 월말 인출 처리 (매월 마지막 거래일)
       portfolio_cash -= monthly_withdrawal
       현금 부족 시 → 부족분만큼 주식 매도로 충당
       주식도 부족 시 → 경고 출력

  ③ 일별 포트폴리오 가치 기록
       일별_가치 = portfolio_cash + (etf_shares × 당일_종가)

[종료] 최종 평가액 + 총 인출액 반환
```

#### ① 배당 재투자

```python
if etf_dividend > 0:
    dividend_amount_etf = etf_shares * etf_dividend       # 배당 수령액
    shares_bought_etf   = dividend_amount_etf / current_etf_price  # 재투자로 추가 매수
    etf_shares += shares_bought_etf
```

배당을 현금으로 쌓지 않고 **즉시 같은 ETF를 추가 매수**하여 복리 효과를 반영합니다.

#### ② 월말 인출 & 주식 매도

```python
# 월의 마지막 거래일 판별
if i == len(dates) - 1:                      # 마지막 날
    is_last_trading_day = True
elif date.month != dates[i+1].month:         # 다음 날이 다음 달이면
    is_last_trading_day = True

if is_last_trading_day:
    portfolio_cash -= monthly_withdrawal      # 현금 인출
    if portfolio_cash < 0:                    # 현금 부족 시 주식 매도
        shares_to_sell = deficit / current_etf_price
        etf_shares -= shares_to_sell
        portfolio_cash += 매도금액
```

- 현금 계좌가 0에서 시작하므로, 첫 번째 인출 시점부터 주식을 매도하게 됩니다.
- `withdrawn_months` 집합으로 **중복 인출을 방지**합니다.

#### ③ 일별 포트폴리오 가치

```python
total_portfolio_value = portfolio_cash + etf_shares * current_etf_price
```

현금 잔고와 ETF 평가액을 합산해 매일 기록합니다.

## 10. 커버드콜 분배금 재투자 전략이 우세한 이유 — 심층 분석

> ⚠️ **분석 오류 정정 (2026-03-17 업데이트)**  
> 이전 버전에서 "KOSPI 200이 약 19–20% 완만히 상승했다"고 기술한 부분은 **잘못된 설명**입니다.  
> 실제 TR ETF (KODEX 200 TR)의 포트폴리오 시뮬레이션 결과(합계 수익률 +135.7%)로 역산하면 해당 ETF 가격은 약 **140% 이상 상승**(약 2.4배)했음을 알 수 있습니다.  
> 이는 **급격한 강세장** 구간에 해당하며, "완만한 상승"이라는 표현은 사실과 다릅니다.  
> 아래 섹션 11에서 실제 시장 국면별 분석을 통해 올바른 해석을 제공합니다.

### 10-1. 두 전략의 수익 구조 비교

| 구분 | KODEX 200 WCC (커버드콜) | KODEX 200 TR (토탈 리턴) |
|------|--------------------------|--------------------------|
| **기초 자산** | KOSPI 200 구성 종목 | KOSPI 200 구성 종목 |
| **추가 수익원** | 콜옵션 매도 프리미엄 | 없음 (주식 배당만) |
| **분배 방식** | 월별 현금 분배 (주식 배당 + 옵션 프리미엄) | 없음 — NAV에 자동 재투자 |
| **상승 참여율** | 제한적 (콜옵션 행사가 이상 상승 포기) | 100% |
| **yfinance Dividends 컬럼** | > 0 (월별 분배금 기록) | = 0 (NAV 내부 재투자) |

### 10-2. 시뮬레이션에서 WCC가 TR을 앞선 3가지 메커니즘

#### ① 옵션 프리미엄이라는 추가 수익원

커버드콜 ETF는 보유 주식 포지션에 콜옵션을 매도하여 **프리미엄을 추가로 수취**합니다.

```
WCC 총 수익 = 주가 상승분 (일부 제한) + 주식 배당금 + 콜옵션 프리미엄
TR  총 수익 = 주가 상승분 (전부)      + 주식 배당금
```

TR이 WCC보다 우위를 가지려면 "TR이 추가로 얻는 주가 상승분" > "WCC가 얻는 옵션 프리미엄"이 되어야 합니다.

#### ② 급등장에서도 옵션 프리미엄이 클 수 있는 이유 — 변동성(Volatility)

이 구간은 KOSPI가 강하게 상승한 동시에 **시장 변동성도 매우 높았습니다**.  
커버드콜 전략에서 옵션 프리미엄은 단순히 시장 방향이 아니라 **변동성 크기**에 연동됩니다.

```
변동성(VIX)이 높을수록 → 옵션 가격(프리미엄) 상승 → WCC 분배금 증가

일반적 통념:   "급등장 = TR 유리"
보완된 이해:   "급등장 + 높은 변동성 = 옵션 프리미엄도 크다 → WCC 분배금도 크다"
```

즉, KOSPI가 크게 오른 이유가 **경기 회복/정책 모멘텀** 등 고변동성 환경이라면,  
WCC는 상승 제한에도 불구하고 큰 프리미엄 수익으로 부분 만회할 수 있습니다.

#### ③ 월별 분배금 재투자의 복리 효과 및 하락 방어

WCC는 매월 분배금을 지급하고, 시뮬레이션은 이를 즉시 재투자합니다.

```
월초 보유 주식 수 × 분배금/주 = 당월 분배 수령액
분배 수령액 ÷ 당월 종가        = 추가 매수 주식 수
```

이 과정이 **13개월 반복**되면서 주식 수가 꾸준히 늘어납니다. 또한 **하락/조정 구간**에서는 WCC가 TR보다 낙폭이 작아 이미 쌓인 수익을 더 잘 지킵니다. 구간별 상세 분석은 섹션 11 참조.

| **강세장 제한 (확인됨)** | 섹션 11에서 확인: 급격한 상승 국면에서는 TR이 WCC보다 수익이 높음 |
| **과거 데이터 의존성** | 이 결과는 2025.03–2026.03의 특정 시장 환경에 기반하며, 미래를 보장하지 않음 |
| **옵션 비용** | 커버드콜 ETF의 운용 수수료는 일반 ETF보다 높을 수 있음 |
| **세금 처리** | 월분배금은 배당소득세 과세 대상, TR은 매도 시점에 세금 발생 — 세후 수익률은 다를 수 있음 |

### 10-5. 투자 전략 시사점

```
✅ 커버드콜(WCC)이 유리한 경우
   - 시장이 횡보하거나 완만히 상승할 때
   - 높은 변동성을 동반한 박스권 또는 완만한 상승장
   - 정기적인 현금 흐름(분배금)이 필요한 투자자
   - 하락 위험을 부분적으로 완충하고 싶을 때

✅ 토탈 리턴(TR)이 유리한 경우
   - 저변동성 + 강력한 일방향 상승장 (낮은 VIX + 강한 상승 추세)
   - 현금 인출이 불필요한 장기 적립식 투자자
   - 세금 이연 효과를 극대화하고 싶을 때 (분배금 없음)
```

## 11. 시장 국면별 시뮬레이션 분석 — 완만한 상승 → 급격한 상승 → 조정

> "KOSPI가 전체적으로 2배 가까이 올랐는데 커버드콜이 이긴 이유는?" — 이 질문에 답하기 위해 전체 기간을 시장 국면별로 분리해 각각 시뮬레이션합니다.

### 11-1. 시장 국면 분류 기준

TR ETF (KODEX 200 TR)의 20일 롤링 수익률을 기준으로 세 국면을 정의합니다.

| 국면 | 조건 | 의미 |
|------|------|------|
| **상승 (급격한 상승)** | 20일 수익률 > +4% | 강한 상승 추세 |
| **하락/조정** | 20일 수익률 < −3% | 명확한 하락 또는 조정 |
| **횡보** | 그 외 | 방향성 없는 박스권 |

연속 동일 국면 블록 중 10거래일 미만인 블록은 인접 블록에 흡수합니다.

### 11-5. 핵심 질문에 대한 답변

> **"KOSPI가 2배 가까이 올랐는데 왜 커버드콜이 이겼나?"**

```
[실제 차트 기반 국면 분석 — 가격 기준]
Phase 1 (완만한 상승, Mar~Sep 2025):  TR +27.1% vs WCC +21.7% → TR 우세 −5.3%p
Phase 2 (급격한 상승, Oct~Feb):       TR +105.8% vs WCC +83.8% → TR 우세 −22.0%p
Phase 3 (조정, Mar 2026):             TR −9.7%  vs WCC −9.2%  → WCC 우세 +0.5%p
──────────────────────────────────────────────────────────────────────────────
가격만 합산 시 TR 우세 (−26.8%p), 그러나...

[실제 yfinance 시뮬레이션 결과 — 분배금 포함]
WCC 합계: 24,513,952 원 (+145.1%)  vs  TR 합계: 23,574,263 원 (+135.7%)  → WCC 우세 +9.4%p
```

**WCC가 전체 기간에서 이긴 이유 — 핵심은 월별 분배금:**

1. **WCC의 월 분배금(옵션 프리미엄 + 배당)이 결정적** — 가격만 보면 TR이 Phase 1·2 모두 우세합니다.  
   실제 WCC 총수익(+145.1%)이 TR(+135.7%)을 앞서는 것은 **매월 지급되는 분배금이 재투자되기 때문**입니다.

2. **변동성 폭등 = 옵션 프리미엄 폭등** — Phase 2(급격한 상승)에서 KOSPI 변동성이 크게 높아져  
   WCC의 콜옵션 매도 수익(프리미엄)이 예년 대비 큰 폭으로 증가했습니다.  
   이 프리미엄 수입이 TR의 가격 우위 −22%p를 상쇄하고도 남았습니다.

3. **분배금의 복리 재투자** — 매달 받는 분배금으로 추가 매수를 통해 주식 수가 늘어나며,  
   이후 상승 국면에서 더 많은 주식을 보유하는 구조가 됩니다.

> ⚠️ **전제 조건**: 이 결과는 **WCC 분배금을 전액 재투자**하는 시뮬레이션 기준입니다.  
> 분배금을 현금으로 지출하면 복리 효과가 줄어들며, TR 우위로 역전될 수 있습니다.  
> 또한 조정 없이 급격한 상승만 지속된다면 TR이 확실히 우세합니다.

> **구간별 정확한 수치 확인**: Google Colab에서 셀 1~8 실행 후 **셀 9를 실행**하면,  
> 실제 278530.KS·498400.KS 가격 및 분배금 데이터 기반으로 국면별 시뮬레이션이 자동 실행됩니다.
